# Contributing

- Branch protection is turned on for main, so all contributions must be made through pull requests and CI must be green before merging.
- Cloudflare is configured to serve all non production branches to preview deployments, so you can test your changes in a live environment before merging.
- Only once merged to main will changes be reflected on the production site, so you can be confident that your changes won't cause any issues for users until they're ready to be released.
